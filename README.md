📖 About CandleMind

CandleMind is an AI-powered stock market learning and fintech education platform designed to help beginners and aspiring traders learn trading through practical experience, real-time market insights, and intelligent AI guidance.

The platform combines trading education, AI mentorship, paper trading, quizzes, live market news, and demat account integrations into one complete ecosystem. CandleMind aims to bridge the gap between financial education and real-world trading by making learning interactive, accessible, and beginner-friendly.

Instead of only watching theoretical videos, users can actively practice trading concepts using simulations, real-time tools, AI assistance, and structured learning modules.

⚙️ Installation
Clone Repository
git clone https://github.com/yourusername/candlemind.git
Navigate to Project
cd candlemind
Install Dependencies
npm install
Start Development Server
npm run dev
🔑 Environment Variables

Create a .env file in the root directory:

# Gemini AI
VITE_GEMINI_API_KEY=your_gemini_api_key

# Finnhub
VITE_FINNHUB_API_KEY=your_finnhub_api_key

# News API
VITE_NEWS_API_KEY=your_news_api_key

# Alpha Vantage
VITE_ALPHA_VANTAGE_API_KEY=your_alpha_vantage_api_key

# Supabase
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
